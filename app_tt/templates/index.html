Memória Brasil

O projeto Memória Brasil objetiva selecionar, digitalizar e transcrever para arquivos editáveis, publicações históricas relevantes para a história estatística do Brasil. Estas publicações, no momento vindas da Biblioteca do Ministério da Fazenda, têm a memória estatística do Brasil nas áreas de demografia, economia, finanças, administração pública, transporte, educação, saúde, entre outras.

Nossa aplicação objetiva contribuir para extrair os dados contidos nas publicações digitalizadas do projeto Memória Estatística e foi desenvolvida da seguinte forma:

Em colabore você poderá escolher com qual publicação deseja contribuir clicando sobre o nome do livro. Após selecionar a publicação alvo você será redirecionado para uma micro-tarefa que compõe um conjunto de tarefas necessárias para a transcrição.

    Existem dois tipos de tarefas que são geradas de acordo com as respostas fornecidas pelos usuários, essas tarefas são:

  • Tarefa de triagem: Nessa tarefa você será apresentado a uma página de um livro. E deverá responder sim caso haja uma tabela naquela página, não caso não haja tabela, ou não sei em caso de dúvida.

  • Tarefa de marcação: Nessa tarefa será apresentado uma página de um livro. Você deverá então delimitar a área das tabelas contidas nessa página utilizando o mouse para desenhar uma moldura sobre a área de cada tabela. Após o desenho de uma moldura, será apresentado um formulário que deverá ser preenchido com as informações solicitadas.

Durante a execução dessas tarefas, você poderá escolher se deseja fazer uma tarefa diferente da que está realizando no momento clicando sobre um botão que aparecerá, sempre que existir uma tarefa diferente disponível.
